# 出差报销税费计算器

一个 Windows 桌面应用，用于计算出差报销中的机票、火车票等交通费用及相关税费。

## 功能特性

- 机票税费计算
- 火车票费用计算
- 退票、改签费用计算
- 直观的图形界面，操作简单

## 技术栈

- **Electron** - 跨平台桌面应用框架
- **React** - 前端 UI 框架
- **Vite** - 构建工具

## 运行环境

- Windows 10 / 11
- 无需额外安装依赖，直接运行即可

## 截图

应用主界面提供了清晰的费用录入和计算功能，帮助快速完成出差报销单填写。
